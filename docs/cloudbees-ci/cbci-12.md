# Cross Team Collaboration External HTTP Endpoints

External HTTP endpoints work with Cross Team Collaboration to enable external systems such as GitHub or Nexus to generate notification events for Pipelines for all the ***managed controllers*** connected to the CloudBees CI Operations Center.
